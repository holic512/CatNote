## 项目概述

**CatNote** 结合了现代前后端技术，为用户提供了一个功能丰富的笔记平台。项目的主要技术栈如下：

- **前端**: Node.JS V20.12.0 + Vue 3 + Vite + Element Plus
- **后端**: Java17 + Spring Boot
- **数据库**: MySQL + Redis
- **富文本编辑器**: TipTap
- **数据展示**: Echarts
- **容器化**: Docker
- **存储方式**: 笔记以 Markdown 格式存储，图片支持本地存储或第三方图床

## 主要功能

### 用户管理

- **用户注册和登录**:
    - 🆕 **注册**：通过邮箱或其他认证方式创建新账户。
    - 🔑 **登录**：通过邮箱,账号和密码登录系统。
    - 🛡️ **身份验证**：使用 JWT（JSON Web Token）进行用户认证。

- **用户个人笔记**:
    - 📓 **个人笔记管理**：用户可以访问和管理自己的笔记。
    - 👤 **用户配置文件**：用户可以编辑自己的个人信息。

### 笔记功能

- ✏️ **创建/编辑笔记**：创建新笔记并进行编辑。
- 🗑️ **删除/归档笔记**：支持笔记分类以便组织和检索。

### 编辑和预览

- 🖋️ **Markdown 编辑器**：支持 Markdown 语法，实时预览 Markdown 内容。
- 👀 **同屏编辑预览**：编辑栏和预览栏同屏显示。

### 导入和导出功能

- **导入功能**:
    - 📥 **Markdown 导入**：支持从本地文件或其他云存储服务中导入 Markdown 格式的笔记。
    - 🔗 **分享链接导入**：通过分享的展示链接导入其他用户的笔记。

- **导出功能**:
    - 📤 **Markdown 导出**：将笔记导出为 Markdown 格式文件，支持单个笔记和批量导出。
    - ⚙️ **导出设置**：提供导出选项，如是否包括图片、标签等。

### 分享功能

- **生成展示链接**:
    - 🔗 **分享设置**：为笔记生成公共展示链接。
    - 🔒 **权限设置**：设置分享链接的权限，如公开、仅限查看等。
    - 📜 **访问控制**：查看分享记录，撤销分享。

- **链接访问**:
    - 🌐 **展示页面**：生成的展示链接将带用户进入一个只读的展示页面。
    - 🔐 **防盗链**：确保分享链接的安全性，防止未经授权的访问。

### 代码高亮

- 💻 **定制代码高亮**：支持多种编程语言的代码高亮显示，自定义高亮规则和样式。
- 🌈 **彩虹方括号**：为括号添加彩虹色显示，增加代码的可读性。

### 图片处理

- 🖼️ **拖放/粘贴图片**：支持直接从剪贴板粘贴或拖放图片到编辑器中，自动上传并插入图片链接。

### 版本控制

- ⏳ **历史版本功能**：记录每次保存的版本，支持查看和恢复历史版本。
- 🔍 **版本比较**：对比不同版本的差异。

### 后台管理功能

- **用户管理**:
    - 👥 **用户管理**：显示所有注册用户的列表,并可以添加,修改,更新,删除用户。

- **笔记管理**:
    - 📋 **笔记列表**：查看所有用户的笔记（管理员权限）。
    - 📜 **笔记审查**：审查和管理笔记内容，删除违规内容。
    - 📊 **统计分析**：查看笔记使用情况统计数据，如创建数量、最活跃用户等。

### 网站配置

- **网站信息**:
    - 🏠 **网站主标题**：设置网站的主要标题，用于浏览器标签和页面头部。
    - 🖋️ **网站副标题**：设置网站的副标题，用于补充主标题，显示在页面中较显著的位置。
    - 🌐 **网站语言**：配置网站的默认语言，以支持多语言环境。

- **SEO 配置**:
    - 🏷️ **Meta 标题**：定义页面在搜索引擎中显示的标题，优化页面的搜索引擎排名。
    - ✍️ **Meta 描述**：简短描述页面内容，用于提高搜索引擎结果页面的点击率。
    - 🔑 **Meta 关键词**：定义页面相关的关键词，帮助搜索引擎理解页面内容。

- **URL 配置**:
    - 🔗 **友好 URL**：启用友好的 URL 结构，提升用户体验和 SEO 效果。
    - 🛠️ **自定义 URL 重写**：配置 URL 重写规则，简化 URL 结构以更符合 SEO 和用户需求。

- **站点地图配置**:
    - 🗺️ **XML 站点地图**：自动生成并维护 XML 格式的站点地图，帮助搜索引擎更好地抓取网站内容。
    - 🔄 **HTML 站点地图**：提供给用户的站点地图，便于导航和快速访问。

- **社交媒体集成**:
    - 🌍 **Open Graph 标签**：优化社交媒体平台的内容预览效果，提高用户点击率。
    - 🐦 **Twitter Cards**：配置 Twitter 卡片，提升 Twitter 上分享内容的视觉效果。
    - 📱 **社交媒体链接管理**：集成多个社交媒体平台的链接，增加网站的社交曝光。

- **分析与跟踪**:
    - 📊 **Google Analytics 集成**：配置 Google Analytics，跟踪网站流量和用户行为。
    - ⏱️ **页面加载时间分析**：监控并优化页面加载速度，提升用户体验。
    - 🔬 **A/B 测试支持**：进行 A/B 测试，以优化网站内容和布局，提高转化率。

- **安全与隐私**:
    - 🔒 **SSL 证书管理**：确保网站的数据传输安全，通过 SSL 证书加密连接。
    - 📜 **隐私政策**：管理隐私政策，确保网站符合数据保护法律和法规。

- **用户体验（UX）优化**:
    - 📱 **响应式设计**：确保网站在各种设备上都能良好显示，提高用户体验。
    - 🧭 **页面导航优化**：提供易于使用的导航菜单和面包屑导航，提升用户访问便捷性。
    - 🚫 **404 页面自定义**：设置自定义的 404 错误页面，帮助用户找到有效内容并返回网站其他部分。
